# 한국어

## 프로젝트 소개

**VALLA**는 자연처럼 편안한 휴식을 콘셉트로 한 홈웨어 브랜드 쇼핑몰 프로젝트입니다.  
브랜드 기획부터 로고, 컬러 시스템, 배너 디자인, UI 구성, 카페24 쇼핑몰 구현까지 전 과정을 직접 제작했습니다.

단순히 쇼핑몰 화면을 만드는 것에서 그치지 않고, 브랜드가 실제 온라인 스토어로 운영될 수 있도록 상품 이미지 방향, 메인 배너 구성, 카테고리 구조, 사용자 흐름까지 함께 설계했습니다.

<br>

## 프로젝트 목표

- 자연스럽고 편안한 홈웨어 브랜드 이미지 구축
- 카페24 기반 쇼핑몰 UI 구현
- 브랜드 콘셉트와 상품 분위기가 연결되는 배너 디자인 제작
- 사용자가 상품을 쉽게 탐색할 수 있는 쇼핑몰 구조 설계
- 1인 제작 포트폴리오로 브랜딩, 디자인, 구현 역량 표현

<br>

## 브랜드 콘셉트

**VALLA**는 계곡과 초원의 고요함에서 영감을 받은 홈웨어 브랜드입니다.  
하루의 끝, 자연 속에서 쉬는 듯한 편안함을 일상에 담아 부드럽고 차분한 홈웨어 경험을 제안합니다.

# English

## Project Overview

**VALLA** is a homewear brand shopping mall project inspired by the concept of natural comfort.  
The entire process was created independently, from brand planning and logo direction to color system, banner design, UI layout, and Cafe24 shopping mall implementation.

This project goes beyond building a simple shopping mall page. It focuses on creating a complete brand experience by connecting visual identity, product mood, category structure, and user flow.

<br>

## Project Goals

- Build a calm and natural homewear brand identity
- Implement a Cafe24-based shopping mall UI
- Design banners that connect the brand concept with product mood
- Create a shopping structure that helps users browse products easily
- Present branding, design, and implementation skills through an independent project

<br>

## Brand Concept

**VALLA** is a homewear brand inspired by the quiet mood of valleys and meadows.  
It suggests a soft and peaceful homewear experience that feels as comfortable as resting in nature at the end of the day.

<br>

## Main Tasks

- Brand naming and concept planning
- Logo and visual identity direction
- Main banner and seasonal banner design
- Product category planning
- Cafe24 shopping mall layout implementation
- HTML and CSS/SCSS styling
- Portfolio project documentation
